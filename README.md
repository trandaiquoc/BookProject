# Development Environment

This project is built and tested in the following environment:

## Server Stack
- **WAMP Server**: 3.3.7 (64-bit)
- **Apache**: Included with WAMP 3.3.7
- **PHP Version**: 8.3.14

## Backend Tools
- **Composer Version**: 2.8.12

## Database Systems
### MySQL
- Provided by WAMP Server

### MongoDB
- **MongoDB Server Version**: 8.2
- **MongoDB PHP Extension**: 2.1.2

## Notes
- The system uses PHP + MySQL as the primary database.
- MongoDB is used for logs, analytics, and supporting metadata.
